# School-DOOMPY

**Een eindwerk / School Final Year Project**

Dit project is een in Python geschreven 3D game-engine, geïnspireerd op de klassieke DOOM (1993). Het is ontworpen en ontwikkeld als afstudeerproject (eindwerk) om te demonstreren hoe de originele DOOM-engine werkte. De basis van dit project maakt gebruik van geavanceerde wiskundige wiskundige technieken zoals Binary Space Partitioning (BSP) bomen en 2D-naar-3D rendering.

## Credits
**Ontwikkelaar & Auteur:** Tayef

## Functionaliteiten
* Volledig functionele rendering van originele DOOM levels (WAD bestanden).
* Ingebouwde navigatie met behulp van een Binary Space Partitioning (BSP) tree.
* Vlotte eerstepersoonscamera met muisbesturing (Mouse Look) en WASD movement.
* Zwaar geoptimaliseerde code met behulp van Pygame en Numba.
* Uitgebreid en duidelijk gedocumenteerde broncode, ontwikkeld met educatieve doeleinden (elke lijn code is voorzien van commentaar).

## Installatie & Gebruik
1. Zorg ervoor dat **Python** is geïnstalleerd op je computer.
2. Installeer de benodigde pakketten via de terminal:
   ```bash
   pip install -r requirements.txt
   ```
3. Start de engine door in de hoofdmap het volgende commando uit te voeren:
   ```bash
   python src/doom_engine.py
   ```

### Besturing

* Kijken: **Muis**
